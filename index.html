🔬 Methodology

🎯

Single-Shot Prompt

Each model received exactly one prompt per test. No retries, no iterations, no agentic tool loops.

🤖

No Agentic Tools

Pure model output — no file system access, no browser tools, no code execution during generation.

🧪

Playwright Scoring

Automated browser testing via Playwright checked functional criteria per app (116 total per model).

📦

Single HTML File

Each output is a self-contained HTML file with inline CSS and JavaScript. No external dependencies.

⏱️

Speed Bonus

Per test, the fastest model earns +2.0 down to +0.5 for the slowest (max +22 across 11 tests).

🌐

Real-World Tasks

11 practical web apps: games, productivity tools, editors, dashboards — things people actually build.
